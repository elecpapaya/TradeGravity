# Support

TradeGravity is a maintainer-led open-source project. Public, reproducible questions help improve the project for everyone.

## Where to ask

- Use a [bug report](https://github.com/elecpapaya/TradeGravity/issues/new?template=bug_report.yml) for reproducible collection, publishing, data, or viewer problems.
- Use a [feature request](https://github.com/elecpapaya/TradeGravity/issues/new?template=feature_request.yml) for a concrete user need or reuse case.
- Use an ordinary GitHub issue for documentation questions or proposed roadmap changes.
- Follow [SECURITY.md](SECURITY.md) for private vulnerability reporting. Never post exploit details or credentials in an issue.

## Helpful information

For data questions, include the reporter ISO3 code, partner, flow, observation period, provider, dataset `generated_at` value, and a link to the upstream record when possible. For software problems, include the command, commit, operating system, Go version, and sanitized logs.

The maintainer reviews reports as capacity allows and cannot promise a response time or individual data analysis. Clear reproductions and focused pull requests are the fastest path to a fix.
